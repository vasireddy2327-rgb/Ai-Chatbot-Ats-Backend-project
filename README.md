AI chatbot + ATS Resume Scorer

## Features
-chatbot using Groq LLM
-ATS Resume scoring system
-FastAPI backend

## Tech Stack
- Python
- FastAPI
- LangChain
- Groq API

## How to Run

1. Clone repository:
   git clone <your-repo-link>

2. Install dependencies:
   pip install -r requirements.txt

3. Create .env file:
   GROQ_API_KEY=your_api_key

4. Run server:
   uvicorn main:app --reload

5. Open browser:
   http://127.0.0.1:8000/docs
